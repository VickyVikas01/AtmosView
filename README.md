AtmosView 🌦️
A sleek, responsive, and real-time weather dashboard that provides live weather data for any city worldwide using the OpenWeatherMap API.

🔗 Live Demo on Netlify

🚀 Features
Global Search: Enter any city name to get instant weather updates.

Real-time Data: Fetches current temperature, humidity levels, and weather descriptions.

Dynamic Visuals: The UI automatically updates with weather-specific emojis and styles based on the condition code (e.g., clear, clouds, rain, snow).

Metric Conversion: Automatically converts temperature from Kelvin (API default) to Celsius.

Error Handling: Built-in validation to handle empty inputs or invalid city names gracefully.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Flexbox & Gradients)

Scripting: JavaScript (ES6+)

API: OpenWeatherMap API

Deployment: Netlify

🏗️ Technical Highlights
Asynchronous Programming: Utilized async/await and the Fetch API to handle network requests without blocking the main thread.

DOM Manipulation: Dynamically created and injected HTML elements based on JSON response data.

Destructuring: Used modern JavaScript destructuring to cleanly extract deeply nested data from API responses.
